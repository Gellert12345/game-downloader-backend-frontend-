Full‑stack game downloader demo with Node.js + Express backend

Game Downloader Web App
A full‑stack demo web application that demonstrates a secure login system and protected file downloads.
The project uses a Node.js + Express backend with session‑based authentication and a vanilla JavaScript frontend.
Logged‑in users can access protected download endpoints, while unauthorized users are blocked.
Technologies used:
– Node.js
– Express
– express‑session
– bcrypt
– Vanilla JavaScript
– HTML / CSS
Note:
This project is a technical demo. Downloaded files are placeholders and do not contain copyrighted content.
